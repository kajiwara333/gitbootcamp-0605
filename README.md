## 10.git rebase -i A~E
A~Eまでを対象に編集操作を行う。
上記コマンドを入力すると、エディタが立ち上がり、
各リビジョンに対しての操作をテキスト形式で指定できる。（Discard、Squashなど）
カードの例で言うと、Eを前に持ってきて、Bを削除、CとDをまとめる、といった操作を指定することとなる。
※コマンドのみで完結しているわけでないためわかりづらい。
